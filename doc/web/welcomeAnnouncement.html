Hi everyone,

Welcome to the course "Learning from data".  We start on Monday Sep. 1 at 13.15 in the HA3 lecture hall, with the first lecture and introduction to the course. 

The course aims to teach an advanced physicist's approach to Learning from data. It is project based, using python and Jupyter Notebooks as a computational environment. There is no written exam but there will be face-to-face discussions with the teachers on your problem set solutions.

Getting ready for the course you are welcome to follow the getting started guide on the course web page. There is a dedicated getting-started session on the first Monday afternoon where the teachers will demonstrate the download of course material and setup of the python environment. After this demonstration you will have a chance to ask questions and get help with your own installation. Note that this is the preferred opportunity for assistance in getting started. The other lab sessions will be focused on exercises, problem sets and projects.

The course is accompanied by a comprehensive set of lecture notes that also includes demonstration and exercise notebooks. These notes are published open-access (https://cforssen.gitlab.io/tif285-book/) in the Jupyter Book format. You are in particular encouraged to get prepared by studying the first three exercise sections that provide guides for the use of Python, Jupyter Notebooks and NumPy (sections 2, 3, and 4 in the Introduction chapter). You are also encouraged to review section 6 that contains a summary of important concepts in mathematical statistics. 

The exercise notebooks can be downloaded  in the ipynb format from the Jupyter Book and can then be run in your newly installed python environment. In case you did not yet manage to complete the getting started installation instructions, you could still be able to run the exercise notebooks online by uploading them at https://colab.research.google.com/. Note, however, that several of the Jupyter Notebook features will not work in the google colab environment.

All lectures and the computer lab sessions will be on campus. Note in particular that the Thursday morning session typically starts with a demonstration or feedback discussion led by one of the teaching assistants. 

Throughout the course, we will be using the Yata Discussions forum to facilitate recurrent group discussions between students and with teachers. See the Online teaching and extra resources.
