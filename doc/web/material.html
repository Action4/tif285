git repository

All files that are associated with this course are available via a public git repository, hosted on https://github.com/physics-chalmers/tif285.git, that you are welcome to clone.

See detailed instructions.

Lecture notes

All lecture notes, exercises and demonstrations are integrated into a Jupyter Book: , written by Christian Forssén under a Creative Commons BY-NC license. These notes are updated continuously. The static html book version is not interactive. However, the source files are available in the git repository. It is recommended to download all the course material using git clone (and to use git pull for possible updates). Alternatively, you can download single notebooks via the jupyter book by clicking on the download button in the upper right corner.

Problem sets

The problem sets consist of Jupyter Notebooks and, in some cases, datafiles. The files are therefore packed together in a gzipped tarball. Extract with, e.g., tar xvfz TIF285_ProblemSet1.tgz

  • Problem set 1: Introduction, linear regression, model validation [tgz]
  • Problem set 2: Bayesian data analysis, Parameter estimation (to be published)
  • Problem set 3: Machine learning methods, Gaussian processes, neural networks (to be published)
