• The project description can be downloaded from the Files page, or found in the doc/pub directory in the course git repository.
  • See deadline on the corresponding Canvas Assignments.
  • Projects are performed in groups of two students; please join a group in Canvas before submission (there are separate groups for each project and you must not work with the same partner).
  • The second part of the project is optional. See examination rules on the course web page.
  • Hand-in your written report (in pdf format) and your solution source code via the Canvas Assignment.
  • Students are allowed to discuss together and help each other when working on the projects. Generative AI tools can be used for coding assistance. However, every student must understand and be able to explain their submitted solution. Plagiarism (of text and/or code) is not allowed. Submissions will be automatically monitored and manually checked.

Written report

  • Page limit: 6 pages (excluding a title page and list of references). 3 extra pages are allowed when doing also the optional extra task.
  • Give a description of the problem and of the methods you have used.
  • Include your results in figures and/or tables with appropriate commentary. All tables and figures should have relevant captions and labels on the axes.
  • Provide an interpretation of your results.
  • For grading and distribution of points we will follow the rubric associated with the Canvas Assignment.
  • Upload the source code of your program as a separate file (.ipynb or .py). Teachers should be able to understand your code with provided commentary.

Late submissions

  • Assignments that are handed in post deadline will be graded at a reduced rate of 2/3 (2 points awarded for every 3 points passed).
  • Late submissions will only be eligible for points corresponding to the minimum required for a passing grade (for Chalmers PhD students this corresponds to grade 4).
  • Late hand-ins will be graded in December/January (handed in before Nov. 30th) or August/September (handed in before July 31st).
